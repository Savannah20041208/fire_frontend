# 彝族火焰纹与 AR 互动数字服饰共创平台

一个展示性的前端项目，展示融合传统彝族文化元素与现代AR技术的数字服饰共创平台。

## 项目特点

- 🎨 **产品设计图展示** - 展示融合彝族火焰纹的多样化服饰设计
- 📊 **服务流程图** - 清晰展示从注册到交付的完整服务流程
- 📱 **AR互动体验** - AR试穿功能界面展示
- 🎯 **现代化UI设计** - 美观的界面和流畅的交互体验

## 技术栈

- React 18
- Vite
- React Router
- CSS3 (现代样式和动画)

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 预览生产构建：
```bash
npm run preview
```

## 功能模块

### 核心展示模块
- **首页** (`/`) - 平台介绍和功能概览
- **产品设计** (`/product`) - 产品设计图展示页面，浏览丰富的服饰设计作品
- **服务流程** (`/service`) - 服务流程图页面，展示从注册到交付的完整流程
- **AR互动** (`/ar`) - AR试穿功能展示页面，实时预览虚拟服饰效果

### 共创与互动模块
- **共创设计** (`/codesign`) - 在线设计工具，选择纹样和服饰类型进行个性化设计
- **社区互动** (`/community`) - 作品分享、话题讨论、设计师展示等社区功能
- **文化知识** (`/culture`) - 彝族文化和火焰纹知识介绍，传统纹样图库
- **用户中心** (`/user`) - 个人资料、我的设计、订单管理、收藏夹等

## 项目结构

```
src/

│   ├── Home.jsx          # 首页
│   ├── ProductDesign.jsx # 产品设计页

│   ├── Community.jsx    # 社区互动页

```

## 注意事项

这是一个纯前端展示项目，不包含后端功能。所有数据都是模拟数据，用于展示界面效果。
